# 安装方法
> 将install.mel拖拽到Maya视窗中，当前工具架会多出一个图标。
> 
> 点击图标即可使用。

# 工具说明
> 本工具用来解决 Maya 与 Houdini 互导，材质丢失问题。

# 插件原理
> 将模型的面转换为Houdini可以识别的面组。
> 
> 获取模型面上的材质，传递给拓扑结构一致的模型。

# 备注
> 一定要勾选 Write Face sets(每次打开都会强制开启）
