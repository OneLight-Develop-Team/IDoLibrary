# 安装方法
> 将install.mel拖拽到Maya视窗中，当前工具架会多出一个图标。
> 
> 点击图标即可使用。

# 工具说明
> ADV动捕数据通过约束切换到TSM绑定上

# 插件原理
> 通过获取ADV和TSM的命名规则
> 从而自动获取到需要的控制器
> 更具控制器的对应关系添加约束

# 使用方法
> + 手动选择控制器，点击 <<< 按钮获取模型
> + 选择一个约束控制器 点击一键获取 来自动识别控制器
> + 获取所有的控制器之后可以点击 一键约束 进行约束
